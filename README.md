# 基于STM32的智能浇花系统

## 项目简介

基于STM32F103C8T6微控制器的智能浇花系统，具备土壤湿度自动检测、智能浇水决策、水位监测报警及远程数据交互功能。

## 功能特性

- **土壤湿度实时采集** — 通过电阻式土壤湿度传感器 + STM32 ADC + DMA 循环采集
- **自动浇水控制** — 湿度低于阈值时自动启动水泵，达到阈值后停止
- **OLED实时显示** — 0.96寸 128×64 OLED，显示湿度/阈值/设置界面
- **按键交互** — 4个独立按键，支持阈值设置、存储、退出等操作
- **水位检测报警** — 水箱水位过低时蜂鸣器 + LED灯双重报警
- **远程监控** — ESP8266 WiFi + OneNet MQTT 云平台，支持手机远程查看与控制
- **微信小程序** — 实时湿度展示、植物预设阈值（栀子花/绿萝/吊兰/自定义）、远程阈值下发
- **掉电保存** — 阈值存储于STM32内部Flash，重启不丢失

## 硬件架构

- **主控芯片**: STM32F103C8T6 (72MHz, 64KB Flash, 20KB SRAM)
- **土壤湿度传感器**: 电阻式模拟输出 (PA1-ADC1_IN1)
- **水位传感器**: DICS-D 数字量输出 (PA2)
- **显示屏**: 0.96寸 OLED 128×64 I2C (PB6-SCL, PB7-SDA)
- **WiFi模块**: ESP8266 (PA9-TX, PA10-RX)
- **水泵**: DC 12V, 1.5-2L/min, NPN三极管驱动 (PB3)
- **电源**: 12V DC → TPS54302DDA (5V) → AMS1117-3.3 (3.3V)
- **报警**: 蜂鸣器 (PB8) + LED (PB9)

## 软件架构

### STM32下位机

前后台架构（裸机），主循环轮询调度：
1. OneNet下行数据接收与解析
2. 按键扫描与阈值设置
3. 水位检测与报警
4. 水泵滞回控制
5. ADC采集与OneNet上报（每1秒）
6. MQTT心跳保活（每30秒）

### 微信小程序

原生微信小程序框架，通过OneNet HTTP API与设备交互：
- 3秒轮询获取设备属性
- 植物预设阈值一键切换
- 滑块/快捷按钮自定义阈值

## 快速开始

### 硬件准备
1. 按原理图制作PCB或使用开发板搭建电路
2. 下载嘉立创文件.epro2 打样PCB
3. 按引脚分配连接各模块

### STM32固件编译
1. 使用Keil MDK打开 `Project/RVMDK（uv5）/BH-F103.uvprojx`
2. 修改 `Onenet/esp8266.c` 中的WiFi信息
3. 修改 `Main/main.c` 和 `Onenet/onenet.c` 中的OneNet产品信息
4. 编译并下载到STM32

### 微信小程序
1. 使用微信开发者工具打开 `微信小程序软件/miniprogram-2/`
2. 修改 `pages/index/index.js` 中的OneNet认证信息
3. 编译运行

## ⚠️ 安全提醒

本仓库中的敏感信息（WiFi SSID/密码、OneNet产品ID与Token等）已替换为占位符 `YOUR_xxx`。使用前请替换为实际值。请勿将真实凭据提交到公开仓库。
